Задачей агента подсчёта времени распада некоторого количества атомов химического элемента является поиск периода полураспада химического элемента и на основе этой информации расчёт времени для распада некоторого количества атомов. Первым аргументом является число атомов химического элемента и вторым аргументом является химический элемент для которого ищутся элементы под отношением период полураспада
