Scp-программа поиска класса сущности. Первым входным параметром данной scp-программы является неизвестный класс сущности, который необходимо определить. Вторым входным параметром является класс, относительно которого будет производиться поиск в глубину. Третьим входным параметром является множество, в которое будет занесёт класс сущности в случае успешного выполнения поиска.
