Задачей агента вычисления числа частиц вещества является поиск всех известных параметров для вычисления числа частиц по соответствующей известным данным формуле. Данный агент инициируется при условии появления в памяти вопросной конструкции, соответствующей запросу вычисления числа частиц вещества. Аргументом запроса является химическое вещество, число частиц которого необходимо вычислить. Результатом выполнения команды является связка отношения число частиц* для данного вещества.
